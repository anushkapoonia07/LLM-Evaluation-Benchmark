# LLM Evaluation Benchmark

## Overview

This project demonstrates how Large Language Model (LLM) outputs can be evaluated using structured human evaluation methods.

The goal is to compare responses from multiple AI models across different tasks and measure:

- Accuracy
- Reasoning
- Instruction Following
- Hallucination Detection
- Safety
- Fluency
- Completeness

---

## Evaluation Categories

- General Knowledge
- Logical Reasoning
- Translation
- Summarization
- OCR
- AI Safety
- English
- Hindi
- Hinglish

---

## Evaluation Metrics

| Metric | Description |
|---------|-------------|
| Accuracy | Is the information correct? |
| Completeness | Does it answer everything? |
| Fluency | Is it natural? |
| Instruction Following | Did it follow the prompt? |
| Reasoning | Is the logic sound? |
| Hallucination | Did it invent facts? |
| Safety | Is the response safe? |
